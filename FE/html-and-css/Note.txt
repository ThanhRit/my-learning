- Thẻ đầu tiên mỗi file html là <!DOCTYPE html>
- Thẻ Head định nghĩa tiêu đề (thẻ title), thẻ meta charset để hỗ trợ mã hoá ngôn ngữ (ví dự UTF-8)
- Thẻ body chứa nội dung của trang web
- Các thẻ thông dụng trong thẻ body
    1. h1 - h6 (heading) thường là tiêu đề (thường 1 file html chỉ dùng 1 thẻ h1)
    2. p - paragraph thẻ nội dung đoạn văn bản
    3. img - image thẻ hình ảnh (thuộc tính src là link ảnh, thuộc tính alt là hiển thị văn bản khi ảnh lỗi)
    4. a - anchor (thẻ đặt liên kết)
    5. ul (unorder list), ol (order list), li (list item) thẻ chứa danh sách sắp xếp hoặc không sắp xếp 
    6. table - thẻ xây dựng bảng (thead - tiêu đề bảng thường đi với th, tbody - nội dung bảng thường đi với tr, td)
    7.input - thẻ đầu vào dữ liệu của user
    8.button - là 1 nút có thể click vào
    9. div - thẻ tạo khối bao quanh
- Attribute các thẻ thường có các thuộc tính bên trong ví dụ thẻ h1 có thuộc tính là title
- Các cách sử dụng css trong html (Cascading Style Sheets)
    + Internal (Viết style thực tiếp thẻ style vào trong file html - có thể viết bất cứ đâu trong file miễn là hợp lệ)
    + External (Viết style vào 1 file .css sau đó trong file html link đến)
    + Inline (viết thẳng vào các thẻ trong file html bằng thuộc tính style)
- CSS Selector là cách gọi 1 element trong html để css
    + id chỉ được dùng duy nhất 1 lần trong file html, khi viết css trỏ đến id thì dùng dấu # cộng với id
    + class được dùng để nhóm nhiều đối tượng, khi viết css trỏ đến class thì dùng dấu . cộng với class
- Thứ tự ưu tiên CSS
    1. Internal và External không có thứ tự ưu tiên, cái nào gọi ra sau thì sẽ override lại 
    2. Inline - 1000
    3. #id - 100
    4. .class - 10
    5. tag - 1
    6. Universal selector (*) trỏ đến tất cả các phần tử trung file html - 0
    7. thẻ con có thể thừa hưởng thuộc tính của thẻ cha - 0
    8. đặt !important > 1000 độ ưu tiên cao nhất

    Điểm càng cao thì độ ưu tiên càng cao

- CSS variable : cách đặt biến css "--" + "tên biến" -> cách dùng var("--" + "tên biến")
- Các đơn vị trong css
    1.Absolute units (đơn vị tuyệt đối)
    +   px (được sử dụng nhiều nhất)
    +   pt
    +   cm
    +   mm
    +   inch
    +   pc
    2. Relative unit (đơn vị trương đối)
    + % tỉ lệ so với thẻ chứa nó ( phụ thuộc vào thẻ chứa nó)
    + rem (phụ thuộc vào thẻ html) tỉ lệ 1-1 với thẻ html
    + em -> phụ thuộc vào thẻ gần nhất chứa nó mà có css tỉ lệ 1-1
    + vw - width , vh - height (viewport) tỉ lệ kích thước chiều ngang và chiều dài của trình duyệt
- Thuộc tính padding là thành phần đệm nằm giữa content của element và border của nó, khi kích thước padding tăng thì kích thước background cũng tăng
    + nếu nhập padding : 10px -> cả 4 hướng đều 10px
    + nếu nhập padding : 10px 20px -> trên dưới 10px, trái phải 20px
    + nếu nhập padding : 10px 12px 8px -> tên 10px, trái phải 12px, dưới 8px
    + nếu nhập padding 10px 12px 8px 14px -> trên 10px, phải 12px, dưới 8px, trái 14px
- Thuộc tính border là thành phần viên của content, nó nằm giữa thành phần padding với thành phần margin, khi tăng kí thước border lên thì kích thước background của content cũng tăng lên
 + có nhiều thuộc tính border-width, border-style, border color
 + với border style là solid thì chiều dày mặc định của border là 2px, vậy nên nếu chỉ set border cho 1 hướng thì phải set width cho các hướng khác là 0
- Thuộc tính margin là khoảng cách nằm bên ngoài của element, khi tăng kích thước của margin thì kích thước background của content sẽ không tăng lên
 + float : left -> các element sẽ nằm ngang theo bên phải
-Thuộc tính box-sizing sẽ giúp chúng ta cân bằng kích thước bao gồm cả border và padding bằng với kích thước đã đặt ban đầu
 + width 100px, height 100px , padding 10px, border width 10px -> tổng kích thước -> content width 60px + height 60px
 + nếu padding và border quá lớn thì content sẽ bị vỡ
- CSS function
    + var() -> Trong CSS, var() biến báo . hàm được sử dụng để chèn giá trị của một thuộc tính tùy chỉnh (còn được gọi là biến CSS) vào khai báo CSS.
    + linear-gradient() -> Hàm gradient() tuyến tính trong CSS tạo ra một hình ảnh bao gồm sự chuyển tiếp mượt mà giữa hai hoặc nhiều màu được chỉ định dọc theo một đường thẳng. Nó thường được sử dụng cho nền
    + rgba() set màu rgba
    + rgb() set màu RGB
    + calc() hàm giúp tính toán
    + attr() dùng để get value của attribute 1 thẻ

- CSS pseudo-class (lớp giả)
 + :root -> lớp giả tham chiếu đến vần tử gốc html
 + :hover -> các thuộc tính sẽ được kích hoạt khi di chuột vào element
 + :active -> các thuộc tính sẽ được kích hoạt khi click chuột vào element
 + first-child -> lấy được thẻ con đầu tiên 
 + last-child -> lấy được thẻ con cuối cùng

- CSS pseudo-elements (phần tử giả) tạo ra các phần tử giả và có thể css cho nó
    + ::before 
    + ::after
    + ::first-letter -> kí tự đầu tiên của element
    + ::first-line -> dòng đầu tiên của element
    + ::selection -> apply css cho element khi chúng ta kéo chuột select vào nó

- Thuộc tính position
    + Relative -> tương đối (element có thể di chuyển mọi nơi, có thể đè lên các element khác)
    + Absolute -> tuyệt đối (element phụ thuộc vào vị trí của element cha có thuộc tính position, và các thẻ con đã khai báo thuộc tính position : absolute thì chỉ được di chuyển vị trí bên trong thẻ cha)
    + Fixed -> phụ thuộc vào khung trình duyệt
    + Sticky -> phụ thuộc vào khung trình duyệt (bám dính) nhưng không khuyến khích dùng bởi vì một số trình duyệt không hổ trợ

- Flex box là kĩ thuật có tính linh động cao
    + Layout khi sử dụng flex box sẽ gồm : Flex container và flex item
    + Layout khi sử dụng flex box sẽ có thuộc tính
        - main axis : trục chính (sẽ có main start và main end) nó sẽ quyết định hướng mà flex item được nằm
        - cross axis : trục vuông góc với trục chính (sẽ có cross start và cross end) sẽ quyết định việc xuống hàng thì flex item sẽ lên trên hay xuống dưới
        - main size là kích thước chính của flex item , song song với main axis
        - cross size là kích thước flex item song song với croos axis
        - main start là phía bắt đầu của flex item, main end thì ngược lại

- Các thuộc tính được sử dụng trong flex box
    + display: flex, inline-flex sẽ quyết định layout có sử dụng flex box hay không
    + flex-direction: row (main axis nằm ngang) | column (main axis nằm dọc) giúp thay đổi phương hướng của main axis, mặc đinh sẽ là row
    + flex-wrap: nowrap(không xuống dòng) | wrap (xuống dòng) | wrap-reverse (không xuống dòng mà nhảy lên trên dòng trên -> đảo chiều cross start và cross end) để chọn kiểu xuống dòng trong flex box
    + flex-basis : <length> điều chỉnh kích thước của main size
    + justify-content (apply cho flex container) : flex-start | flex-end | center | space-between | space-around giúp căn chỉnh vị trí của flex-item theo main axis
    + justify-self (apply cho flex item) : flex-start | flex-end | center tương tự như justify-content nhưng apply cho riêng flex item
    + align-content (apply cho flex container) : flex-start | flex-end | center giúp căn chỉnh vị trí của flex-item theo cross axis
    + align-self (apply cho flex item) : flex-start | flex-end | center tương tự align-content nhưng apply cho mỗi flex item
    + flex-grow : <number> thay đổi kích thước main size của flex item giúp tăng, kích thước này rất linh động
    + flex-shrink : <number> thay đổi kích thước main size của flex item giúp giảm, kích thước này rất linh động
    + flex:<number> cú pháp viết tắt cho flex-basis, flex-grow, flex-shrink
    + order : quyết định thứ tự của flex item
    + flex-flow: cú pháp viết tắt của flex-direction và flex-wrap

- Trong flex container có nhiều flex item mà chỉ có 1 flex item có flex >= 1 thì flex item đó sẽ cố gắng chiếm hết không gian theo chiều main axis
- CSS BEM (Block Element modifier)
    + là tiêu chuẩn đặt tên class khi viết css
    + (Block Element modifier) -Block : khối, Element: phần tử nằm trong khối, modifier: Bổ sung ý nghĩa cho block và element
    + Vì sao nên dùng BEM :
        - Tránh mỗi người đặt một kiểu
        - Tránh members đặt class trùng nhau, CSS đè lên nhau
    + Cú pháp:
        - .block
        - .block__element
        - .block--modifier
        - .block__element--modifier

- Cú pháp .class + .class sẽ áp dụng selector cho các element kế bên cũng tên class (không áp dụng cho element đầu tiên)
